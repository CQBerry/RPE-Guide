# 参与贡献

我希望与更多人一起维护本指南，你可以对文章中不合理的部分进行修改、可以补充我描述不全面的部分、你甚至可以写一个新的页面！只要理由足够合理，我就会听取你的建议。

  1. Fork 本仓库；
  2. 修改页面或撰写新页面（页面中请全部使用相对路径）；

提示

如有图片请使用 AVIFopen in new window 格式，并放置在 docs/.vuepress/public/assets/imgs/contents/ 下。

如果是撰写新页面，请同时在 docs/.vuepress/sidebar.ts 中添加该页面的索引（格式请参考 sidebaropen in new window），并在页面的 Frontmatter 部分中添加相应的 前后页面链接open in new window。

  1. 提交 Pull requestopen in new window，并说明修改的理由（或者新页面的作用）；
  2. 等待 review，如有修改意见，请按照修改意见修改；
  3. 大功告成 🥰！

# 图片问题

本文档因图片较多，为压缩图片体积，图片均采用 AVIF 格式存储，若无法加载图片，请参考 caniuseopen in new window 更换浏览器。

如果您是 Windows 8 及以上用户，请在 Microsoft Store 中下载 AV1 视频扩展open in new window。

# 离线阅读

推荐使用离线阅读，目前图片的加载速度还不够优秀。

git clone https://github.com/CQBerry/RPE-Guide.git --depth 1

cd RPE-Guide

yarn install # 安装依赖，

yarn dev # 开启本地服务器，默认端口 8080。
